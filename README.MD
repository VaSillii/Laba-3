# ЛАБОРАТОРНАЯ РАБОТА №3

### Общая задача:

Необходимо создать программу, которая будет зашифровать данные или расшифровывать их при помощи шифра Виженера.

### Требования:
1. Программа должна осуществлять дешифровку информации, находящейся в файле и вывод её на экран пользователя.
2. Программа должна предоставлять возможность для зашифровывания информации, введенной пользователем с клавиатуры, а также возможность указать ключ.
3. Программа должна предоставлять возможность сохранить дешифрованную информацию в отдельный файл, с указанием его названия и директории для сохранения
4. Основной функционал должен быть покрыт автоматическими Unit тестами, с использованием стандартных возможностей Unit Testing Framework.  

### Реализовано: 
1. ASP.Net MVC приложение;
2. Xamarin Android приложение;
3. Написаны Unit-тесты основного функционала ASP.Net приложения.
